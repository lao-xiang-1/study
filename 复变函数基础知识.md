# 复变函数基础知识：拉普拉斯变换 (Laplace Transform)

在自动控制原理中，拉普拉斯变换（拉氏变换）是将常微分方程转化为代数方程的重要工具。以下是常见的拉氏变换公式及定理。

## 1. 常见函数的拉氏变换对

设时间函数为 $f(t)$ ($t \ge 0$)，其拉氏变换为 $F(s) = \mathscr{L}[f(t)]$。

| 信号名称 | 时域函数 $f(t)$ | 复频域函数 $F(s)$ |
| :--- | :--- | :--- |
| 单位脉冲函数 | $\delta(t)$ | $1$ |
| 单位阶跃函数 | $1(t)$ 或 $u(t)$ | $\frac{1}{s}$ |
| 单位斜坡函数 | $t$ | $\frac{1}{s^2}$ |
| 抛物线函数 | $t^2$ | $\frac{2}{s^3}$ |
| $n$ 次幂函数 | $t^n$ | $\frac{n!}{s^{n+1}}$ |
| 指数衰减函数 | $e^{-at}$ | $\frac{1}{s+a}$ |
| 正弦函数 | $\sin(\omega t)$ | $\frac{\omega}{s^2 + \omega^2}$ |
| 余弦函数 | $\cos(\omega t)$ | $\frac{s}{s^2 + \omega^2}$ |
| 衰减正弦函数 | $e^{-at}\sin(\omega t)$ | $\frac{\omega}{(s+a)^2 + \omega^2}$ |
| 衰减余弦函数 | $e^{-at}\cos(\omega t)$ | $\frac{s+a}{(s+a)^2 + \omega^2}$ |

## 2. 拉氏变换的基本性质与定理

- **线性定理**：
  $$\mathscr{L}[a \cdot f_1(t) \pm b \cdot f_2(t)] = a \cdot F_1(s) \pm b \cdot F_2(s)$$

- **延时定理 (时域平移)**：
  如果时间延迟了 $t_0$ ($t_0 > 0$)，则：
  $$\mathscr{L}[f(t - t_0)] = e^{-s t_0} F(s)$$

- **位移定理 (复频域平移)**：
  如果在时域乘以指数衰减函数 $e^{-at}$，则：
  $$\mathscr{L}[e^{-at} f(t)] = F(s + a)$$

- **微分定理**：
  $$\mathscr{L}[f'(t)] = s F(s) - f(0)$$
  零初始条件下($f(0)=0$)，微分运算等效于在复频域乘以 $s$。

- **积分定理**：
  $$\mathscr{L}[\int_0^t f(\tau) d\tau] = \frac{F(s)}{s}$$
  零初始条件下，积分运算等效于在复频域除以 $s$。

- **初值定理** (用于求时间 $t \to 0$ 时的值)：
  $$\lim_{t \to 0} f(t) = \lim_{s \to \infty} s F(s)$$

- **终值定理** (用于求稳态误差或稳态值，条件是极点在 s 平面左半部或原点)：
  $$\lim_{t \to \infty} f(t) = \lim_{s \to 0} s F(s)$$
