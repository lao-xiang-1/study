# 常见指令助记（8051）

## 数据传送

- `MOV`：把源操作数送到目的操作数。
- `MOV A,#data`：把立即数送入累加器 A。
- `MOV A,Rn`：把寄存器 Rn 的内容送入 A。
- `MOV Rn,A`：把 A 的内容送入寄存器 Rn。
- `MOV direct,A`：把 A 的内容送到直接地址单元。
- `MOV A,direct`：把直接地址单元内容送入 A。

## 算术运算

- `ADD`：A 与源操作数相加，结果存入 A。
- `ADDC`：A 与源操作数及进位位 CY 相加，结果存入 A。
- `SUBB`：A 减去源操作数及借位位 CY，结果存入 A。
- `INC`：操作数加 1。
- `DEC`：操作数减 1。
- `MUL AB`：A 与 B 相乘，结果低 8 位在 A，高 8 位在 B。
- `DIV AB`：A 除以 B，商在 A，余数在 B。

## 逻辑运算

- `ANL`：按位与运算。
- `ORL`：按位或运算。
- `XRL`：按位异或运算。
- `CLR`：清零（可用于 A、位变量或进位位）。
- `CPL`：按位取反（可用于 A、位变量或进位位）。

## 控制转移

- `SJMP`：短相对无条件跳转。
- `LJMP`：长地址无条件跳转。
- `JZ`：当 A 为 0 时跳转。
- `JNZ`：当 A 不为 0 时跳转。
- `JC`：当进位位 CY=1 时跳转。
- `JNC`：当进位位 CY=0 时跳转。
- `DJNZ`：操作数减 1 后，若不为 0 则跳转。

## 子程序与返回

- `LCALL`：长地址调用子程序。
- `ACALL`：绝对地址调用子程序（页内调用）。
- `RET`：子程序返回。
- `RETI`：中断服务程序返回。

## 位操作

- `SETB bit`：把指定位设置为 1。
- `CLR bit`：把指定位清为 0。
- `CPL bit`：把指定位取反。
- `JB`：指定位为 1 时跳转。
- `JNB`：指定位为 0 时跳转。
