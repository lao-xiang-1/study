8051单片机中的特殊功能寄存器（SFR）是直接映射硬件功能的核心标识符，物理地址固定在80H~FFH范围内。这些符号代表寄存器的直接地址名称（如ACC对应E0H），与汇编指令中的寄存器简写（如A）存在关键区别：ACC用于需要显式地址的操作（如PUSH ACC），而A仅用于寄存器寻址指令（如ADD A, #30H）。以下为最常用的12个核心SFR速查表（按地址升序排列，排除52系列扩展寄存器）：

一、基础运算与控制寄存器
1. 累加器（ACC/A）
- 物理地址：E0H  
- 核心功能：  
  - 算术逻辑运算的核心操作数，绝大多数指令默认使用。  
  - A是寄存器寻址简写（如MOV A, R0），ACC是直接地址标识（如PUSH ACC）。  
- 复位值：00H  
- 关键特性：  
  - 可位寻址（地址E0H能被8整除），其位地址范围为E0H~E7H。  

2. B寄存器
- 物理地址：F0H  
- 核心功能：  
  - 专用于乘除法运算：MUL AB结果存于A（低8位）和B（高8位）；DIV AB商存A、余数存B。  
- 复位值：00H  
- 关键特性：  
  - 可位寻址（地址F0H能被8整除），位地址范围为F0H~F7H。  

3. 程序状态字（PSW）
- 物理地址：D0H  
- 核心功能：  
  - 存储运算状态标志，关键位包括：  
    - CY（PSW.7）：进位/借位标志。  
    - AC（PSW.6）：辅助进位标志（低4位向高4位进位时置1）。  
    - OV（PSW.2）：有符号数溢出标志。  
    - P（PSW.0）：ACC中"1"的奇偶标志。  
- 复位值：00H  
- 关键特性：  
  - 可位寻址（地址D0H能被8整除），位地址范围为D0H~D7H。  

二、数据与堆栈管理寄存器
1. 堆栈指针（SP）
- 物理地址：81H  
- 核心功能：  
  - 指向当前堆栈栈顶地址，堆栈采用向上生长机制（压栈时先SP+1再存数据）。  
- 复位值：07H  
- 关键特性：  
  - 不可位寻址（地址81H不能被8整除），必须初始化到30H以上以避免覆盖工作寄存器区。  

2. 数据指针（DPTR）
- 物理地址：DPL=82H（低8位），DPH=83H（高8位）  
- 核心功能：  
  - 16位地址指针，用于访问外部RAM（MOVX指令）或程序存储器（MOVC查表指令）。  
- 复位值：0000H  
- 关键特性：  
  - 不可位寻址，通常以16位形式联合使用（如MOV DPTR, #1000H）。  

三、I/O与外设控制寄存器
1. I/O端口寄存器（P0~P3）
- 物理地址：  
  - P0=80H，P1=90H，P2=A0H，P3=B0H  
- 核心功能：  
  - 控制4个8位并行I/O端口的输入/输出状态（写寄存器控制引脚电平，读寄存器获取外部信号）。  
- 复位值：FFH（所有引脚默认高电平）  
- 关键特性：  
  - 全部可位寻址（地址均以0或8结尾），例如P1.0的位地址为90H。  

2. 定时器/计数器控制寄存器
(1) TCON（定时器控制）
- 物理地址：88H  
- 核心功能：  
  - 控制定时器启停（TR0/TR1）和中断标志（TF0/TF1）。  
- 复位值：00H  
- 关键特性：  
  - 可位寻址（地址88H能被8整除），位地址范围为88H~8FH。  

(2) TMOD（定时器模式）
- 物理地址：89H  
- 核心功能：  
  - 设置定时器工作模式（4种模式）和功能选择（定时/计数）。  
- 复位值：00H  
- 关键特性：  
  - 不可位寻址（地址89H不能被8整除），需整体操作。  

3. 中断控制寄存器
(1) IE（中断使能）
- 物理地址：A8H  
- 核心功能：  
  - 全局使能/禁用中断（EA位）和各中断源开关（如EX0控制外部中断0）。  
- 复位值：00H（所有中断默认关闭）  
- 关键特性：  
  - 可位寻址（地址A8H能被8整除），位地址范围为A8H~AFH。  

(2) IP（中断优先级）
- 物理地址：B8H  
- 核心功能：  
  - 设置中断源优先级（高/低两级），解决中断冲突。  
- 复位值：00H（所有中断默认低优先级）  
- 关键特性：  
  - 可位寻址（地址B8H能被8整除），位地址范围为B8H~BFH。  

4. 串行通信控制寄存器
(1) SCON（串行控制）
- 物理地址：98H  
- 核心功能：  
  - 配置串口工作模式（4种模式）和控制收发使能（REN）。  
- 复位值：00H  
- 关键特性：  
  - 可位寻址（地址98H能被8整除），位地址范围为98H~9FH。  

(2) SBUF（串行数据缓冲）
- 物理地址：99H  
- 核心功能：  
  - 发送/接收数据的统一缓冲区（写操作存发送数据，读操作取接收数据）。  
- 复位值：不定  
- 关键特性：  
  - 不可位寻址，物理上分为发送和接收两个独立缓冲器。  

四、关键使用注意事项
1. A与ACC的区别：  
   - A仅用于寄存器寻址指令（如ADD A, R0），ACC用于需显式地址的操作（如PUSH ACC）。  
   - 错误示例：PUSH A（非法指令），正确应为PUSH ACC。  

2. 复位状态：  
   - SP默认指向07H，需手动初始化到30H以上（避免覆盖工作寄存器区00H~1FH）。  
   - IE默认全关闭，必须通过SETB EA等指令启用中断。  

3. 位寻址规则：  
   - 仅地址能被8整除的SFR可位寻址（如PSW=D0H，P1=90H），共11个寄存器支持。  
   - 位操作指令（如SETB P1.0）直接操作单个引脚，无需读-改-写端口寄存器。  

4. 物理地址唯一性：  
   - SFR地址与内部RAM高128字节（80H~FFH）重叠，但仅21个离散地址有效，其他地址读写无意义。  

此表格覆盖了8051单片机开发中最关键的SFR，实际使用时需结合具体指令要求区分寄存器简写（如A）与直接地址标识（如ACC）。
