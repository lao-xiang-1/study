# 自动控制原理易忘知识点

## 1. 传递函数的定义与公式
- **定义**：在**零初始条件**下，系统**输出**的拉氏变换除以**输入**的拉氏变换。
- **公式**：$$G(s) = \frac{C(s)}{R(s)}$$
  *(其中 $C(s)$ 为输出，$R(s)$ 为输入)*

## 2. 闭环系统的传递函数
对于标准负反馈系统（前向传递函数为 $G(s)$，反馈传递函数为 $H(s)$）：
- **闭环传递函数**：$$\Phi(s) = \frac{G(s)}{1 + G(s)H(s)}$$正反馈的话分母是减号。
- *注：若是单位负反馈（$H(s) = 1$），则 $\Phi(s) = \frac{G(s)}{1 + G(s)}$*

## 3. 单位阶跃函数
- **时域表达式**：
  $$r(t) = \begin{cases} 1, & t \ge 0 \\ 0, & t < 0 \end{cases}$$
- **复频域 (拉氏变换)**：
  $$R(s) = \frac{1}{s}$$
