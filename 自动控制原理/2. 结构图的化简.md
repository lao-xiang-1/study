# 结构图的化简

## 1. 结构图化简的目标
结构图（方框图）用于描述系统各环节之间的信号关系。化简的目标是把复杂结构等效为一个总传递函数：
$$G_{eq}(s)=\frac{C(s)}{R(s)}$$

## 2. 基本连接方式
- **串联**：前级输出作为后级输入。
  $$G_{eq}(s)=G_1(s)G_2(s)\cdots G_n(s)$$

- **并联**：各环节输入相同，输出代数相加。
  $$G_{eq}(s)=G_1(s)\pm G_2(s)\pm \cdots \pm G_n(s)$$

- **反馈连接**：
  - 负反馈：
    $$G_{eq}(s)=\frac{G(s)}{1+G(s)H(s)}$$
  - 正反馈：
    $$G_{eq}(s)=\frac{G(s)}{1-G(s)H(s)}$$

## 3. 比较点（求和点）与引出点（分支点）移动规则
- **比较点前移/后移**：移动后需在旁路中补偿相应环节，保持进入各支路的信号不变。
- **引出点前移/后移**：移动后需在新支路上补乘或补除相应传递函数，保证引出信号等效。

## 4. 常见化简步骤（考试通用）
1. 先化最内层小回路（优先消去局部反馈）。
2. 再做串联、并联合并。
3. 处理比较点和引出点移动，尽量形成标准反馈结构。
4. 重复以上步骤，直到只剩单输入单输出。
5. 最后检查符号：正负号、反馈方向、增益是否遗漏。
