
### 方法一：时域经典法（求解齐次微分方程）

这种方法的核心是解常系数线性齐次微分方程。

**步骤：**
1. **写出齐次微分方程**：
   将系统微分方程的右端（激励项）置为零。例如，原方程为 $y''(t) + a_1 y'(t) + a_0 y(t) = f'(t) + b_0 f(t)$，置零后得到：
   $$y_{zi}''(t) + a_1 y_{zi}'(t) + a_0 y_{zi}(t) = 0$$

2. **列出特征方程并求特征根**：
   将微分算子替换为 $\lambda$（或 $r$），得到特征方程：
   $$\lambda^2 + a_1 \lambda + a_0 = 0$$
   解出特征根 $\lambda_1, \lambda_2, \dots, \lambda_n$。

3. **写出零输入响应的通解形式**：
   根据特征根的性质写出 $y_{zi}(t)$ 的表达式：
   - **单实根** $\lambda$：对应项为 $C e^{\lambda t}$
   - **$k$ 重实根** $\lambda$：对应项为 $(C_1 + C_2 t + \dots + C_k t^{k-1})e^{\lambda t}$
   - **一对共轭复根** $\alpha \pm j\beta$：对应项为 $e^{\alpha t}(C_1 \cos(\beta t) + C_2 \sin(\beta t))$

4. **代入初始状态求待定系数**：
   利用题目给定的初始状态 $y_{zi}(0_-), y_{zi}'(0_-), \dots$（注意：零输入响应中，由于没有外部激励，状态在 $t=0$ 时不发生跳变，即 $y_{zi}(0_+) = y_{zi}(0_-)$）。
   将 $t=0$ 代入通解及其导数，解出系数 $C_1, C_2 \dots$，从而得到最终的 $y_{zi}(t)$。

### 方法二：s 域求解法（单边拉普拉斯变换法）

这是最推荐、最不易出错的方法，尤其是在初始状态较复杂时。它将微积分运算转换为代数运算。

**步骤：**
1. **对微分方程两边取单边拉普拉斯变换**：
   利用拉普拉斯变换的微分性质，并将输入项 $f(t)$ 设为 $0$。
   **核心性质（含初始状态）：**
   - $\mathscr{L}[y(t)] = Y(s)$
   - $\mathscr{L}[y'(t)] = sY(s) - y(0_-)$
   - $\mathscr{L}[y''(t)] = s^2Y(s) - s y(0_-) - y'(0_-)$

2. **代入初始条件并化简**：
   将已知的初始状态 $y(0_-), y'(0_-)$ 具体数值代入上式。

3. **求解 $Y_{zi}(s)$**：
   把含有 $Y(s)$ 的项放在等式左边，把由初始状态产生的常数或含 $s$ 的项移到等式右边。
   解出零输入响应的拉普拉斯变换 $Y_{zi}(s)$，它通常是一个有理分式。

4. **部分分式展开与拉普拉斯逆变换**：
   对 $Y_{zi}(s)$ 进行部分分式展开（查表法），然后逐项求逆变换，得到时域表达式 $y_{zi}(t)$。
   最后的结果通常要乘以单位阶跃信号 $U(t)$，表示响应从 $t=0$ 开始。

---

### 💡 总结对比
- **时域法**物理意义直观，适合低阶系统，但求待定系数的过程在阶数较高时解方程组较繁琐。
- **s 域法（拉氏变换）**代数化运算，天然包含了初始状态的影响，无需手动匹配 $0_+$ 和 $0_-$ 的跳变关系，是《信号与系统》解题中最常用且最标准的方法。
