在控制工程、信号与系统以及常微分方程的学习中，拉普拉斯逆变换（$\mathscr{L}^{-1}$）是将复频域（$s$ 域）表达式还原回时域（$t$ 域）的核心工具。

*注意：以下公式均基于单边拉普拉斯变换，即默认时间 $t \ge 0$ 时信号存在（通常隐含乘以阶跃函数 $u(t)$）。*

---

### 一、 常用基本函数变换对（查表用）

这是最基础的“字典”，我们在做逆变换时，最终目的就是把复杂的 $F(s)$ 拆解成下面这些基本形式。

| 时域函数 $f(t)$ ($t \ge 0$) | $s$ 域函数 $F(s)$ | 物理/信号意义 |
| :--- | :--- | :--- |
| $\delta(t)$ | $1$ | **单位冲激信号**（瞬间极大的敲击） |
| $1$ 或 $u(t)$ | $\frac{1}{s}$ | **单位阶跃信号** / 常数（突然打开的开关） |
| $t$ | $\frac{1}{s^2}$ | **斜坡信号**（随时间匀速增加） |
| $\frac{t^{n-1}}{(n-1)!}$ | $\frac{1}{s^n}$ | 高阶多项式信号 ($n$ 为正整数) |
| $e^{-at}$ | $\frac{1}{s+a}$ | **指数衰减**信号 ($a>0$ 时系统稳定) |
| $t \cdot e^{-at}$ | $\frac{1}{(s+a)^2}$ | 带重根的衰减信号（临界阻尼系统常见） |
| $\frac{t^{n-1}}{(n-1)!} e^{-at}$ | $\frac{1}{(s+a)^n}$ | 高阶重根衰减信号 |
| $\sin(\omega t)$ | $\frac{\omega}{s^2 + \omega^2}$ | **正弦波**（等幅振荡，无阻尼） |
| $\cos(\omega t)$ | $\frac{s}{s^2 + \omega^2}$ | **余弦波**（等幅振荡，无阻尼） |
| $e^{-at}\sin(\omega t)$ | $\frac{\omega}{(s+a)^2 + \omega^2}$ | **衰减正弦波**（欠阻尼系统，逐渐平息的振荡） |
| $e^{-at}\cos(\omega t)$ | $\frac{s+a}{(s+a)^2 + \omega^2}$ | **衰减余弦波**（欠阻尼系统） |

### 二、 拉普拉斯逆变换的核心性质（运算法则）

当 $F(s)$ 比较复杂时，不能直接查表，必须依靠以下性质将其化简或变形。已知 $\mathscr{L}^{-1}\{F(s)\} = f(t)$：

#### 1. 线性性质（最常用）
加减法和常数倍数可以直接拆分提取。
$$ \mathscr{L}^{-1} \{ aF_1(s) + bF_2(s) \} = a f_1(t) + b f_2(t) $$
*(我们在上一个问题中把分式拆开就是用了这个性质)*

#### 2. 微分性质（求导法则）
*   **在 $s$ 域乘以 $s$**：相当于在时域中**求导**。
    *   若初始条件为零（求传递函数时）： $\mathscr{L}^{-1}\{ sF(s) \} = f'(t)$
    *   若初始条件不为零： $\mathscr{L}^{-1}\{ sF(s) - f(0) \} = f'(t)$
    *   推而广之： $\mathscr{L}^{-1}\{ s^n F(s) \} = f^{(n)}(t)$ （零初始条件下）

#### 3. 积分性质（积分法则）
*   **在 $s$ 域除以 $s$**（即乘以 $\frac{1}{s}$）：相当于在时域中**求积分**。
    $$ \mathscr{L}^{-1} \left\{ \frac{1}{s} F(s) \right\} = \int_{0}^{t} f(\tau) d\tau $$
    *(这正是你一开始问的“积分环节”的数学依据)*

#### 4. $s$ 域平移性质（第一平移定理）
*   **在 $s$ 域将 $s$ 替换为 $(s+a)$**：相当于在时域中乘以衰减因子 **$e^{-at}$**。
    $$ \mathscr{L}^{-1} \{ F(s+a) \} = e^{-at} f(t) $$
    *(应用场景：当遇到分母是 $s^2 + 4s + 13$ 时，通常需要“配方”成 $(s+2)^2 + 3^2$，然后利用此定理求出带有衰减的振荡 $e^{-2t}\sin(3t)$ )*

#### 5. $t$ 域平移性质（第二平移定理 / 延迟定理）
*   **在 $s$ 域乘以 $e^{-as}$**：相当于在时域中发生了**时间延迟（滞后）**。
    $$ \mathscr{L}^{-1} \{ e^{-as} F(s) \} = f(t-a) \cdot u(t-a) $$
    *(应用场景：控制系统中的纯滞后环节，例如管道运输延迟)*

#### 6. $s$ 域微分性质
*   **在 $s$ 域对 $s$ 求导**：相当于在时域乘以 **$-t$**。
    $$ \mathscr{L}^{-1} \left\{ \frac{d}{ds}F(s) \right\} = -t \cdot f(t) $$

### 💡 实战解题技巧：部分分式展开法（Partial Fraction Expansion）

在实际做题（比如求解微分方程或求系统阶跃响应）时，你得到的 $F(s)$ 往往是一个复杂的分式，比如：
$$ F(s) = \frac{3s + 2}{s^2 + 3s + 2} $$
字典表里没有这个公式。你必须使用**部分分式展开**，把它拆成最简形式：
1.  对分母进行因式分解：$s^2 + 3s + 2 = (s+1)(s+2)$
2.  设参拆分：$\frac{3s + 2}{(s+1)(s+2)} = \frac{A}{s+1} + \frac{B}{s+2}$
3.  求出 $A=-1, B=4$
4.  得到：$F(s) = \frac{-1}{s+1} + \frac{4}{s+2}$
5.  最后查表（使用线性性质+指数衰减公式）：$$f(t) = -e^{-t} + 4e^{-2t}$$
