**题目信息：**
系统微分方程：$y''(t) + 3y'(t) + 2y(t) = f(t)$
激励信号：$f(t) = -2e^{-t}U(t)$
初始状态：$y(0^-) = 1, y'(0^-) = 2$
求解：零输入响应 $y_{zi}(t)$、零状态响应 $y_{zs}(t)$ 和全响应 $y(t)$。

推荐使用**单边拉普拉斯变换**来求解，过程如下：

### 1. 求零输入响应 $y_{zi}(t)$
零输入响应是指在没有外加激励（即 $f(t)=0$）的情况下，仅由系统初始状态引起的响应。
对微分方程左边取单边拉普拉斯变换，带入初始状态，右边置为0：
$$[s^2Y_{zi}(s) - sy(0^-) - y'(0^-)] + 3[sY_{zi}(s) - y(0^-)] + 2Y_{zi}(s) = 0$$
将 $y(0^-) = 1, y'(0^-) = 2$ 代入：
$$s^2Y_{zi}(s) - s - 2 + 3sY_{zi}(s) - 3 + 2Y_{zi}(s) = 0$$
合并同类项，提取 $Y_{zi}(s)$：
$$(s^2 + 3s + 2)Y_{zi}(s) = s + 5$$
$$Y_{zi}(s) = \frac{s+5}{s^2+3s+2} = \frac{s+5}{(s+1)(s+2)}$$
使用部分分式展开：
$$Y_{zi}(s) = \frac{A}{s+1} + \frac{B}{s+2}$$
其中：
$A = \left. \frac{s+5}{s+2} \right|_{s=-1} = \frac{4}{1} = 4$
$B = \left. \frac{s+5}{s+1} \right|_{s=-2} = \frac{3}{-1} = -3$
所以，$Y_{zi}(s) = \frac{4}{s+1} - \frac{3}{s+2}$
进行拉普拉斯逆变换，得到**零输入响应**：
$$y_{zi}(t) = (4e^{-t} - 3e^{-2t})U(t)$$

---

### 2. 求零状态响应 $y_{zs}(t)$
零状态响应是指在初始状态为零（即 $y(0^-)=0, y'(0^-)=0$）的情况下，仅由外加激励引起的响应。
先求激励信号的拉普拉斯变换：
$$F(s) = \mathcal{L}\{-2e^{-t}U(t)\} = \frac{-2}{s+1}$$
对方程两边取拉普拉斯变换（初始状态设为0）：
$$(s^2 + 3s + 2)Y_{zs}(s) = F(s)$$
$$Y_{zs}(s) = \frac{F(s)}{(s+1)(s+2)} = \frac{-2}{(s+1)^2(s+2)}$$
对包含重根的分式进行部分分式展开：
$$Y_{zs}(s) = \frac{C}{s+2} + \frac{D}{s+1} + \frac{E}{(s+1)^2}$$
通分得：$C(s+1)^2 + D(s+1)(s+2) + E(s+2) = -2$
- 令 $s = -1$，得到 $E(1) = -2 \Rightarrow E = -2$
- 令 $s = -2$，得到 $C(-1)^2 = -2 \Rightarrow C = -2$
- 比较 $s^2$ 的系数：$C + D = 0 \Rightarrow D = -C = 2$

所以，$Y_{zs}(s) = \frac{-2}{s+2} + \frac{2}{s+1} - \frac{2}{(s+1)^2}$
进行拉普拉斯逆变换，得到**零状态响应**（注意最后一项使用了 $t$ 乘性质）：
$$y_{zs}(t) = (-2e^{-2t} + 2e^{-t} - 2te^{-t})U(t)$$

---

### 3. 求全响应 $y(t)$
全响应等于零输入响应与零状态响应之和：
$$y(t) = y_{zi}(t) + y_{zs}(t)$$
$$y(t) = (4e^{-t} - 3e^{-2t}) + (-2e^{-2t} + 2e^{-t} - 2te^{-t})$$
合并同类项，得到最终的**全响应**：
$$y(t) = (6e^{-t} - 5e^{-2t} - 2te^{-t})U(t)$$
