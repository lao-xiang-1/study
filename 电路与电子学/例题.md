**✅ 正确答案：D**

### 题目分析与详细解答

**电路结构判断**：  
观察电路图可知，电阻 $R$、电感 $L$、电容 $C$ **三者并联**在 $a$、$b$ 两端。因此这是一个**并联电路**。

---

### 正确解法（推荐使用导纳）

因为是并联电路，各支路两端电压相同，总电流为各支路电流之和，所以**先求导纳**更方便。

1. 各元件导纳：
   - 电阻导纳：$\dot{Y}_R = \frac{1}{R}$
   - 电感导纳：$\dot{Y}_L = \frac{1}{j\omega L} = \frac{1}{j\omega L}$
   - 电容导纳：$\dot{Y}_C = j\omega C$

2. 总导纳（并联导纳直接相加）：
   $$
   \dot{Y}_{ab} = \dot{Y}_R + \dot{Y}_L + \dot{Y}_C = \frac{1}{R} + \frac{1}{j\omega L} + j\omega C
   $$

3. 等效阻抗为导纳的倒数：
   $$
   Z_{ab} = \frac{1}{\dot{Y}_{ab}} = \frac{1}{\dfrac{1}{R} + \dfrac{1}{j\omega L} + j\omega C}
   $$

**此结果与选项 D 完全一致**。

---

### 各选项分析

| 选项 | 表达式 | 判断 | 说明 |
|------|--------|------|------|
| **A** | $R + j\omega L - j/\omega C$ | 错误 | 这是**串联**时的阻抗表达式 |
| **B** | $1/(R + j\omega L - j/\omega C)$ | 错误 | 分母错误地把阻抗直接相加后取倒数 |
| **C** | $1/R + 1/j\omega L + j\omega C$ | 错误 | 这是**总导纳** $\dot{Y}_{ab}$，不是阻抗 $Z_{ab}$ |
| **D** | $1/(1/R + 1/j\omega L + j\omega C)$ | **正确** | 正确地先求并联总导纳，再取倒数得到阻抗 |

---

### 知识点小结

- **并联电路**优先使用**导纳**（Y并 = Y1 + Y2 + Y3），再取倒数得阻抗。
- **串联电路**优先使用**阻抗**（Z串 = Z1 + Z2 + Z3）。
- 本题中三个元件并联在 $a$、$b$ 之间，因此必须用导纳求解。
