# 个人学习笔记整理

个人向的学习笔记整理，只包含学校课程，无其他自学内容。
涵盖本学期重点课程内容，未来可能拓展到其他课程。

**全部为markdown格式**，该格式对ai友好，可配合ai学习。
许多内容为**ai生成**，虽然都经过我审核，但是还可能出现错误，请自行判断。

所有笔记力求简短，只记录最重要的信息，减少阅读量

目前还很乱，预计我复习的时候会整理好

---

## 使用说明：

虽然主要目的是 为了帮助记忆+督促自己学习，但是既然已经开源，姑且附上建议的使用方法

1. 搭配[obsidian](https://obsidian.md/)：一个本地markdown阅读器。可下载第三方插件，如obsidian-copilot，配置apikey 以使用ai。
2. 自己创建一个git仓库：上课的时候大多不方便带电脑，那就得在手机上记笔记。创建git仓库可实现双端同步。
3. 自行修改内容：笔记是为了帮助记忆，那肯定是自己写的 最适合自己，最好能自行增添内容。

---

## 推荐工具：
1. [markitdown](https://github.com/microsoft/markitdown)：一个可将其他格式的文件如pdf，word 转成markdown，只是转换出来一般有问题，最好配合ai修改。
2. [obsidian-sync](https://github.com/pourmand1376/Obsidian-Sync)：基于termux实现安卓端git仓库同步。
